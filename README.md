# MadGraph Processes Repository

This repository contains 13 different MadGraph processes with their corresponding cards.

## Structure
- `Cards/` - Contains all process directories
- Each process has standard MadGraph card files

## Process Cards
1. proc_card.dat - Process definition
2. param_card.dat - Model parameters
3. run_card.dat - Run settings
4. shower_card.dat - Parton shower
5. madspin_card.dat - Spin correlations
